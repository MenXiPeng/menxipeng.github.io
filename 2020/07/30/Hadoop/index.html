Hadoop 3.x 单机安装

发布于：Jul 30, 2020

1.3k words

7 min

关于hadoop 3.x 版本基础安装个人总结，建议以官网为指导进行配置，不同版本不一致，本教程适用于3.1.x
官网地址：https://hadoop.apache.org/docs/current/

一、下载hadoop
https://hadoop.apache.org/releases.html
二、解压hadoop包
tar -zxvf hadoop-3.1.2.tar.gz
三、修改hosts文件
1. vim /etc/hosts  
2. 添加 0.0.0.0(内网ip) master master 
3. source /etc/profile
四、生成ssh密钥
1. ssh-keygen -t rsa -P '' 
# 第一个回车-输入要生成的文件名(默认id_rsa)
# 第二个回车-输入生成密钥密码(默认为空)
# 第三个回车-重复输入密码(默认为空)
2. cat /root/.ssh/id_rsa.pub >> /root/.ssh/authorized_keys #master主机(互信密钥)
# 不设置密钥互信会导致启动时出现权限不足的异常
五、配置hadoop Home
1. vim /etc/profile 
2. 添加 #hadoop enviroment hadoop环境变量
export HADOOP_HOME=/usr/local/hadoop-3.1.2/ #hadoop 根目录
export PATH="$HADOOP_HOME/bin:$HADOOP_HOME/sbin:$PATH"
export HADOOP_CONF_DIR=$HADOOP_HOME/etc/hadoop
export YARN_CONF_DIR=$HADOOP_HOME/etc/hadoop 
3. source /etc/profile #重新加载源文件
六、修改hadoop 配置文件
  • $HADOOP_HOME # hadoop 根目录
  • $HADOOP_HOME/etc/hadoop/hdfs-site.xml #添加如下内容
  • $HADOOP_HOME/etc/hadoop/mapred-site.xml(cp mapred-site.xml.template mapred-site.xml)
  • $HADOOP_HOME/etc/hadoop/yarn-site.xml
    七、格式化namenode 节点
    hadoop namenode -format
八、脚本配置
1.启动
#!/bin/bash
echo -e "\033[31m ========Start The Cluster======== \033[0m"
echo -e "\033[31m Starting Hadoop Now !!! \033[0m"
/opt/hadoop-2.7.3/sbin/start-all.sh
echo -e "\033[31m Starting Spark Now !!! \033[0m"
/opt/spark-2.1.0-bin-hadoop2.7/sbin/start-all.sh
echo -e "\033[31m The Result Of The Command \"jps\" :  \033[0m"
jps
echo -e "\033[31m ========END======== \033[0m"

2.停止脚本
#!/bin/bash
echo -e "\033[31m ===== Stoping The Cluster ====== \033[0m"
echo -e "\033[31m Stoping Spark Now !!! \033[0m"
/opt/spark-2.1.0-bin-hadoop2.7/sbin/stop-all.sh
echo -e "\033[31m Stopting Hadoop Now !!! \033[0m"
/opt/hadoop-2.7.3/sbin/stop-all.sh
echo -e "\033[31m The Result Of The Command \"jps\" :  \033[0m"
jps
echo -e "\033[31m ======END======== \033[0m"
九、查看节点 jps
----- Jps
----- DataNode
----- NameNode
----- NodeManager
----- SecondaryNameNode
看到所有节点代表启动成功
十、root启动报错
start-dfs.sh / stop-dfs.sh
HDFS_DATANODE_USER=root
HADOOP_SECURE_DN_USER=hdfs
HDFS_NAMENODE_USER=root
HDFS_SECONDARYNAMENODE_USER=root

start-yarn.sh / stop-yarn.sh
YARN_RESOURCEMANAGER_USER=root
HADOOP_SECURE_DN_USER=yarn
YARN_NODEMANAGER_USER=root
参考 http://www.cnblogs.com/purstar/p/6293605.html
